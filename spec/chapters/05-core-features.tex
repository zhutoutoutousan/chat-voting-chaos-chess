\chapter{Core Features}

\section{Feature Overview}

The Chat Voting Chaos Chess Platform includes comprehensive features comparable to chess.com, enhanced with unique interactive elements. This chapter details all core features and their specifications.

\section{User Management}

\subsection{User Registration \& Authentication}

\begin{itemize}
    \item \textbf{OAuth Integration}: Support for Google, GitHub, Discord
    \item \textbf{Email/Password}: Traditional registration option
    \item \textbf{Email Verification}: Required for account activation
    \item \textbf{Password Reset}: Secure password recovery flow
    \item \textbf{Profile Management}: User profile creation and editing
    \item \textbf{Avatar Upload}: Profile picture management
\end{itemize}

\subsection{User Profiles}

\begin{itemize}
    \item \textbf{Public Profile}: Display username, avatar, ELO rating, statistics
    \item \textbf{Profile Statistics}: Games played, win/loss/draw records
    \item \textbf{Achievement System}: Badges and accomplishments
    \item \textbf{Activity History}: Recent games and activity
    \item \textbf{Social Features}: Follow/unfollow other players
\end{itemize}

\section{Chess Game Features}

\subsection{Game Modes}

\begin{itemize}
    \item \textbf{Classic Chess}: Standard chess gameplay
    \item \textbf{Chaos Mode}: Chat voting influences moves
    \item \textbf{Blitz}: Fast-paced games (3+0, 5+0, 10+0)
    \item \textbf{Rapid}: Medium time control (15+10, 30+0)
    \item \textbf{Classical}: Long time control (60+0, 90+30)
    \item \textbf{Correspondence}: Multi-day games
    \item \textbf{Custom Time Controls}: User-defined time limits
\end{itemize}

\subsection{Game Features}

\begin{itemize}
    \item \textbf{Move Validation}: Legal move checking
    \item \textbf{Move History}: Complete game notation (PGN)
    \item \textbf{Game Analysis}: Post-game analysis with engine
    \item \textbf{Takeback Requests}: Request move takeback (if enabled)
    \item \textbf{Draw Offers}: Offer and accept/decline draws
    \item \textbf{Resignation}: Surrender option
    \item \textbf{Time Management}: Clock display and management
    \item \textbf{Game Replay}: Review completed games
\end{itemize}

\subsection{Chess Engine Integration}

\begin{itemize}
    \item \textbf{Stockfish Integration}: Engine analysis
    \item \textbf{Move Suggestions}: Engine recommendations
    \item \textbf{Position Evaluation}: Position strength analysis
    \item \textbf{Best Move Calculation}: Engine best move
    \item \textbf{Depth Analysis}: Configurable analysis depth
\end{itemize}

\section{ELO Rating System}

\subsection{Rating Calculation}

\begin{itemize}
    \item \textbf{Initial Rating}: Starting ELO (typically 1200 or 1500)
    \item \textbf{Rating Updates}: After each rated game
    \item \textbf{K-Factor}: Variable K-factor based on rating
    \begin{itemize}
        \item New players (< 30 games): K = 40
        \item Intermediate (30-100 games): K = 20
        \item Established (> 100 games): K = 10
    \end{itemize}
    \item \textbf{Expected Score}: Calculated using ELO formula
    \item \textbf{Rating Change}: Based on actual vs expected score
\end{itemize}

\subsection{Rating Categories}

\begin{itemize}
    \item \textbf{Overall Rating}: Combined rating across all time controls
    \item \textbf{Blitz Rating}: Separate rating for blitz games
    \item \textbf{Rapid Rating}: Separate rating for rapid games
    \item \textbf{Classical Rating}: Separate rating for classical games
    \item \textbf{Correspondence Rating}: Rating for correspondence games
    \item \textbf{Puzzle Rating}: Rating for puzzle solving
\end{itemize}

\subsection{Rating Display}

\begin{itemize}
    \item \textbf{Rating History}: Graph showing rating over time
    \item \textbf{Rating Distribution}: Percentile ranking
    \item \textbf{Rating Milestones}: Achievement notifications
    \item \textbf{Leaderboards}: Top players by rating
\end{itemize}

\section{Chat Voting Chaos Mode}

\subsection{Core Concept}

Chaos Mode allows spectators to vote on moves, creating an interactive and unpredictable gameplay experience.

\subsection{Voting Mechanics}

\begin{itemize}
    \item \textbf{Spectator Voting}: Viewers can vote on suggested moves
    \item \textbf{Move Suggestions}: Engine or popular move options
    \item \textbf{Voting Window}: Time-limited voting period
    \item \textbf{Vote Aggregation}: Most popular move is played
    \item \textbf{Tie Breaking}: Random selection or engine preference
    \item \textbf{Vote Weighting}: Optional premium user vote weighting
\end{itemize}

\subsection{Chat Integration}

\begin{itemize}
    \item \textbf{Live Chat}: Real-time chat during games
    \item \textbf{Move Suggestions}: Chat commands for move suggestions
    \item \textbf{Moderation}: Chat moderation tools
    \item \textbf{Emotes}: Chess-themed emotes and reactions
    \item \textbf{User Mentions}: @mention functionality
\end{itemize}

\subsection{Chaos Mode Features}

\begin{itemize}
    \item \textbf{Public Games}: Games open to spectator voting
    \item \textbf{Private Games}: Invite-only chaos games
    \item \textbf{Voting History}: Record of all votes and outcomes
    \item \textbf{Chaos Rating}: Separate rating for chaos mode
    \item \textbf{Popular Games}: Highlighted chaos games
\end{itemize}
